# 🐾 Peddy

> My first JavaScript project with API integration.

Peddy is a responsive pet adoption web application that fetches and displays real-time pet data from an external API.

---

## 🔗 Links

- Live Demo: https://sagorteasc-peddy.vercel.app/
- GitHub: https://github.com/sagorteasc/Peddy

---

## 🌐 Project Overview

Peddy allows users to:

- Browse pets dynamically
- View pet details
- Filter pets by category
- Interact with real API data

---

## 🚀 Features

- 🔗 API integration
- 📡 Fetch dynamic data using JavaScript
- ⏳ Loading state handling
- 📂 Category filtering
- 📱 Fully responsive design

---

## 🧠 What I Learned

While building this project, I practiced:

- Fetching data from an API
- Using `async/await`
- Handling promises
- DOM manipulation with dynamic data
- Error handling basics
